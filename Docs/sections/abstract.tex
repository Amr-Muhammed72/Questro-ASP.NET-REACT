\chapter*{\centering {\small {Abstract}}}
\noindent This project introduces Questro, an intelligent hybrid recommendation platform for both games and movies. Questro combines collaborative filtering,content-based filtering, and social discovery features to deliver personalized, diverse, and easily discoverable recommendations. The platform aggregates rich metadata from multiple sources (genres, platforms, gameplay mechanics,cast, ratings) and employs dimensionality-reduction
techniqu    es (e.g., SVD) together with clustering to mitigate sparsity and
improve recommendation quality. To address cold-start challenges,\\ Questro uses hybrid initialization strategies including onboarding questionnaires, metadata-driven heuristics, and user clustering.
\\
\\
\noindent The core recommendation pipeline is extended with social features — shared watchlists/playlists, community ratings, and highlighted reviews to increase trust, engagement, and serendipitous discovery while reducing decision fatigue. Preliminary experiments against baseline models demonstrate measurable improvements in recommendation accuracy and user satisfaction.
\\
\\
\noindent Questro provides a unified platform that transforms content discovery from a solitary algorithmic task into a socially enriched, user-centered experience.
